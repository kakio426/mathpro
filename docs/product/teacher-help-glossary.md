# Teacher Help Glossary

수학프로의 교사용 화면에서는 개발자 용어를 그대로 노출하지 않고, 수업 준비 관점의 쉬운 표현을 함께 제공한다.

## 화면 용어 기준

| 화면 용어 | 쉬운 표현 | 설명 기준 |
| --- | --- | --- |
| HTML | 움직이는 수업자료 원본 | AI가 만들어 준 한 장짜리 자료 파일이다. 선생님이 직접 코딩한다는 의미로 설명하지 않는다. |
| AI 요청문 | AI에게 보내는 제작 주문서 | Gemini 같은 AI에게 원하는 학년, 개념, 활동을 알려주는 글이다. |
| 안전 검사 | 학생 화면 보호 확인 | 외부 연결, 파일 업로드, 권한 요청처럼 수업자료에 필요 없는 기능을 막는 절차다. |
| 조작 기록 | 학생이 만진 과정 | 선택, 드래그, 힌트, 재시도, 제출, 완료를 시간 흐름으로 남긴 기록이다. |
| 참여 코드 | 로그인 없는 입장 번호 | 학생이 계정 없이 수업자료에 들어가기 위한 짧은 코드다. |

## UX 원칙

- 어려운 용어는 숨기지 말고 바로 옆에 쉬운 표현을 붙인다.
- 설명은 “기술이 무엇인가”보다 “선생님 수업에서 어떤 의미인가”로 쓴다.
- 오류나 경고 문구는 탓하는 말 대신 다음 행동을 알려준다.
- 학생에게 보이는 화면에는 제작 도구, HTML, postMessage 같은 단어를 노출하지 않는다.
